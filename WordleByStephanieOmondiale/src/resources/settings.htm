Game Mode Settings

The application supports two game modes:

  • Random Mode: In this mode, the game selects a random word from the word list, and the player must guess it within six attempts.
  • Absurdle Mode: This mode dynamically adjusts the word list to keep the game challenging. The player has six attempts to narrow down and guess the word.

To choose a mode, use the appropriate command-line parameters or select the mode in the GUI.
